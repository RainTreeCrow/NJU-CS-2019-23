# Java实现画图工具
## 项目简述
大三上学期设计模式课程设计，被强行要求使用大量设计模式。画图工具在画笔模式下可以绘制线段、三角形、长方形、圆形等多种图形，设置画笔粗细、颜色、透明度等；在选择模式下，可选择画板上已绘制的图形，对其进行移动、放大、缩小、删除等操作。
